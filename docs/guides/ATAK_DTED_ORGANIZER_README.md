# ATAK DTED Organizer README

## Purpose
Copies DTED files into state folders while preserving subfolders.

## Supported naming
- `w101/n29.dt2`
- `n30w103.dt2`

## Default states file
```text
/media/paul/ExtraDrive/Map/DTED2/us_states_simple.geojson
```

## Run
```bash
python3 organize_dted_by_state_finalbuild.py
```

## Example
```bash
python3 organize_dted_by_state_finalbuild.py /media/paul/ExtraDrive/Map/DTED2 /media/paul/ExtraDrive/Map/DTED_by_state
```
